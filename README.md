# Doubloons — Pirate Currency

Static landing: lore, Solana note, contract copy.

## Deploy on Vercel

1. Push this repo to GitHub.
2. [vercel.com](https://vercel.com) → **Add New Project** → Import the repository.
3. Framework: **Other** (no build). Root: `.` → **Deploy**.

Or CLI: `npx vercel --prod`

## Local preview

Open `index.html` in a browser or:

```bash
npx serve .
```
